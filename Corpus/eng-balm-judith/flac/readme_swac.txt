
 SWAC Audio Collection - Readme file
 -----------------------------------

 This package is maintained by the Shtooka Project.
 It is available as .tar packages at:

 http://packs.shtooka.net/

 Ogg and Flac audio files contain SWAC fields encapsulated in Vorbis 
 Comment tags. Indexes of SPX and MP3 packages are generated using
 information encapsulated in corresponding FLAC audio files.

 For more information about SWAC fields, please visit the SWAC Audio 
 Collection Homepage at:

 http://shtooka.net/swac/ 

 For more information about the Shtooka Project, please visit the Shtooka 
 Project Homepage at:

 http://shtooka.net/
